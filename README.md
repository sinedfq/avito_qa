<h1>Тестовое задание Avito QA-Инженер</h1>

<h3>Используемые технологии:</h3>

1. Python
2. Pytest
3. Requests
4. Postman
5. re
6. sys

<h3>Используемые техники тест-дизайна:</h3>

1. Эквивалентное разделение
2. Анализ граничных значений 
3. Тестирование состояний и переходов
4. Позитивное и негативное тестирование
5. Тестирование ошибок 
6. Тестирование API
7. Тестирование обязательных полей
8. Тестирование типов данных
9. Тестирование бизнес-логики
10. Сценарное тестирование

<h3>Для запуска приложения необходимо: </h3>

1. Устанавливаем зависимости: ```pip install -r requirements.txt```
2. Выполняем команду ```python run_tests.py```

<h3>Описание файлов проекта: </h3>

1. run_tests.py - файл для запуска всех авто тестов
2. test_delete_requests.py - файл, в котором содержаться запросы на удаление (DELETE)
3. test_get_requests - файл, в котором содержаться запросы на получение (GET)
4. test_post_requests - файл, в котором содержаться запросы на создание/отправку (POST)
5. conftest.py - специальный файл в pytest для хранения общих настроек и фикстур
6. TESTCASES.md - файл, в котором содержаться тест-кейсы
7. BUGS.md - файл, в котором содержаться баг-репорты
8. avito_first_task.pdf - файл с первым заданием
